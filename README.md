# Guitar Fretboard Trainer

ギターの指板上の音名・度数を覚えるためのWebアプリ。

## 機能

### 指板マップ
- 全ポジションの音名/度数を一覧表示
- 度数表示ではルート音を選択可能（ルートは赤、各度数は色分け）
- 練習範囲の絞り込みに連動（範囲外は薄く表示）

### クイズ（3モード）
- **ポジション → ノート**: 指板上のハイライト位置の音名を回答
- **ノート → ポジション**: 表示された音名の位置を指板上でタップ
- **度数**: ルート音に対するハイライト位置の度数を回答

### 学習サポート
- **練習範囲の絞り込み**: 弦単位・フレット範囲で出題範囲を限定
- **プリセット**: 「開放〜4F」「5〜9F」「6弦のみ」など
- **ヒント**: クイズ中に開放弦からの半音数、5F=隣弦の開放音などの目印を表示
- **スコア**: 正解率・連続正解・最高記録（localStorageで永続化）
- **シャープ/フラット表記切替**

## 技術スタック

- React 19 + TypeScript
- Vite 8
- Tailwind CSS 3
- SVGで指板を描画（外部ライブラリなし）

## セットアップ

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
npm run preview
```

## おすすめの学習手順

1. 「指板マップ」で全体像を把握する
2. 範囲を「6弦のみ・0〜4F」に絞ってクイズに挑戦
3. 慣れたら範囲を広げる（5弦追加、フレット範囲拡大）
4. 度数モードでインターバルも覚える
